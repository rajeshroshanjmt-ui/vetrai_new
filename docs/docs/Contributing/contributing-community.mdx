There are several ways you can interact with the Vetrai community and learn more about the Vetrai codebase.

## Join the Vetrai Discord server

Join the [Vetrai Discord Server](https://discord.gg/EqksyE2EX9) to ask questions and showcase your projects.

## Follow Vetrai on X

Follow [@vetrai_ai](https://twitter.com/vetrai_ai) on X to get the latest news about Vetrai.

## Star Vetrai on GitHub

If you like Vetrai, you can star the [Vetrai GitHub repository](https://github.com/vetrai-ai/vetrai).
Stars help other users find Vetrai more easily, and quickly understand that other users have found it useful.

Because Vetrai is open-source, the more visible the repository is, the more likely the codebase is to attract [contributors](/contributing-how-to-contribute).

## Watch the GitHub repository

You can watch the [Vetrai GitHub repository](https://github.com/vetrai-ai/vetrai) to get notified about new releases and other repository activity.

To get release notifications only, select **Releases only**.

If you select **Watching**, you will receive notifications about new releases as well as issues, discussions, and pull requests, if you are interested in that activity.
For information about customizing repository notifications, see the [GitHub documentation on repository subscriptions](https://docs.github.com/en/account-and-profile/managing-subscriptions-and-notifications-on-github/managing-subscriptions-for-activity-on-github/viewing-your-subscriptions).

## Request enhancements and get help through GitHub

You can also submit feature requests and get help with Vetrai through the GitHub repository.
For more information, see [Get help and request enhancements](/contributing-github-issues).
